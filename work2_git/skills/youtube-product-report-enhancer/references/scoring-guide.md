# 评分解读指南

把 YouTube 分数理解为内容需求信号，不要理解为销量信号。

- `youtube_search`：搜索覆盖度。多个种子搜索下都出现相关视频时更强。
- `youtube_velocity`：每日观看速度。适合判断近期动能，但可能被单条爆款视频扭曲。
- `youtube_engagement`：点赞和评论占播放量的比例。用户提问、讨论或争论时更强。
- `youtube_comments`：抽样评论痛点密度。评论出现 mess、smell、stress、replacement、sizing、cleaning、leaking、renter constraints 或重复抱怨时更强。
- `creator_diversity`：独立频道数量。多个创作者独立覆盖同一产品类型时更强。

排序建议：

- 有创作者扩散且评论痛点强的产品应上调。
- 只有单条视频爆火的产品应下调。
- 产品定义模糊的聚类应下调或改成“观察”。
- 需要医疗、食品、保健品或治疗宣称的产品应下调或排除。
- 分数只是输入之一，最终排序要看电商可测试性。
